# Data description

1. Minute bar and second bar OHLCV data for dozens of stocks
2. Time ranges around their earning report release events
3. For each stock and each earning event, both minute bar and second bar data are provided, minute bar data is about a few days before and after the earning report release event, second bar data is before, at and after the earning report release event

# File location

1. /Volumes/Extreme SSD/trading_data/stock/data/ltf_ohlcv is the root folder for all the data
2. a metadata folder is under the root folder which includes the metadata of all the stocks
